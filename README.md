To build and run development code
---------------------------------
1) make html && make server

To push to production
---------------------
cd output
git add .
git commit -m "misc content update"
git push -u origin master

To push content to github source control
----------------------------------------
cd ~/ghpages
git add .
git commit -m "misc commit"
git push -u origin master
